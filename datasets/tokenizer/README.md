# Tokenizers

This folder stores tokenizer metadata files (`meta.pkl`), which define:

- vocabulary (`stoi`, `itos`)
- token mappings
- special tokens

These files are generated during tokenization and reused for:

- training
- sampling
- reinforcement learning

Example:

- `meta_chembl.pkl`
- `meta_crossdocked.pkl`

⚠️ Not included in the repository by default.
They can be regenerated by running the tokenization pipeline.
